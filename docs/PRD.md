# 需求说明

## 功能概述

此软件是一款记录摩托车油耗的工具。用户可以借助手机拍摄加油机的信息，然后调用OCR，识别出金额、升数、单价三个内容。

## 模块规划

此软件分为两个大模块：用户未录入信息模块、用户录入信息后模块
### 新老用户模块
新用户：
1.添加摩托车信息、摩托车交强险、摩托车年检信息
2.选择摩托车、录入加油信息

老用户：
1.展示摩托车信息，以及当前加油周期的油耗
2.修改加油信息、修改摩托车信息

### 交互路径
1. 空数据模块
用户首先进行摩托车添加，之后可对摩托车信息进行修改。
摩托车信息维护内容：车辆名称、购买日期

用户未进行加油记录，进入此软件展示提示记录加油信息的界面。

用户可以通过表单录入、也可以选择通过图片走 OCR 识别录入。

### UI 设计描述

设计理念：符合移动端的用户操作习惯。交互流畅可理解，及时反馈。配色自然舒适。配色尽量使用 tailwind 的 lime。

如果用户有多个车辆，需要通过顶部 tab 进行滑动切换。

每个车辆卡片展示 车辆名称、当前加油信息、里程数、下次保养日期或里程数据

尽可能把UI设计的像行驶证或身份卡类似这种感觉的。


## 代码实现
1. 采用 React 19 + react router + tailwindcss + shadcn + zustand
2. 本地 OCR 组件 https://github.com/naptha/tesseract.js#tesseractjs
